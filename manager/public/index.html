Quick Actions
Quick Start
1
Login
Click Launch Login in the top-right corner. A terminal opens with the Codex login command that works on this machine.
2
Capture the new session
After the login flow finishes, return to the app and press Capture Session. That imports the current ~/.codex/auth.json into the account registry.
3
Refresh quota
Use Refresh Quota whenever you want the latest usage data for the active account.
4
Switch accounts
Go to Accounts and click Switch on the account you want to activate. The app updates both auth.json and the local registry.
What Each Tab Does
Dashboard
Shows the current active account, average quota health, best account recommendation, plan breakdown, and runtime paths.
Accounts
Lets you search by email or alias, filter by plan, isolate low-quota accounts, and switch accounts directly from the table.
Settings
Controls quota refresh mode and background switching behavior. It also keeps the main quick actions in one place.
How to use
Contains the in-app guide for new users and teammates so they do not need to remember the workflow.
Feature Guide
Launch Login
Starts the Codex login flow in a terminal. Use this whenever you want to sign in to a new Codex account.
Capture Session
Reads the latest current session from ~/.codex/auth.json and saves it as a managed account snapshot.
Refresh Quota
Runs a foreground quota refresh through codex-auth. The active account usage snapshot becomes current in the UI.
Switch
Activates another stored account. This is the main way to move between imported accounts without manual file editing.
Settings Explained
Usage API
When enabled, the app asks codex-auth to use the ChatGPT usage API for more current quota data. This is more accurate but can carry more account risk.
Auto Switch
Turns on the background switch logic from codex-auth. When thresholds are crossed, the tool can move to a better account automatically.
Low Quota Filter
In the Accounts tab, this filter shows accounts below 20% remaining in either the 5h or weekly window so you can avoid them quickly.
Plan Filters
These chips narrow the table to a specific plan type such as PLUS or any other plan detected in the registry.
